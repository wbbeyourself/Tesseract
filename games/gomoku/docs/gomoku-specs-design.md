# 五子棋网页版设计文档

**日期：** 2026-05-31
**项目：** ClaudeZone / gomoku
**状态：** 已批准

---

## 概述

一个纯前端的五子棋（Gomoku）网页游戏，支持人机对战和双人对弈。棋盘规格 40×40，采用传统木纹视觉风格，AI 提供简单/中等/困难三个难度等级。

## 游戏模式

### 人机对战

- 玩家执黑先行，AI 执白
- AI 难度三档可选：简单、中等、困难
- 游戏开始前玩家可输入自己的名字
- 左侧面板显示 AI 信息，右侧面板显示玩家信息

### 双人对战

- 玩家 1 执黑（左），玩家 2 执白（右）
- 游戏开始前弹出输入框，分别输入双方名字
- 左侧面板显示玩家 1，右侧面板显示玩家 2
- 两人共用同一屏幕轮流落子

## 棋盘

- **规格：** 40×40 网格
- **形状：** 严格正方形（Canvas 宽高相等，每个格子也是正方形）
- **渲染：** HTML5 Canvas
- **自适应：** 根据窗口高度自动缩放，始终保持 1:1 比例

## AI 引擎

### 算法：启发式评分 + 模式匹配

对棋盘每个空位在 4 个方向（横、竖、左斜、右斜）扫描连子模式，累计得分：

| 模式 | 分数 | 说明 |
|------|------|------|
| 连五 | 1,000,000 | 直接获胜 |
| 活四 | 100,000 | 两头开放的四连 |
| 冲四 | 10,000 | 一头被堵的四连 |
| 活三 | 5,000 | 两头开放的三连 |
| 眠三 | 500 | 一头被堵的三连 |
| 活二 | 200 | 两头开放的二连 |
| 眠二 | 50 | 一头被堵的二连 |

评分同时考虑进攻分（AI 连子）和防守分（对手连子），取较大值。

### 三档难度

| 难度 | 策略 |
|------|------|
| 简单 | 单步最高分 + 10% 随机扰动 |
| 中等 | 单步评分 + 1 层对手反击预判 |
| 困难 | Minimax 搜索深度 2（AI 看自己 2 步 + 对手 1 步），Alpha-Beta 剪枝，无随机扰动 |

### 性能优化（40×40 棋盘）

- **候选位筛选：** 只评估已有棋子周围 2 格内的空位（约 50-150 个候选位）
- **增量评估：** 每步只重新计算受影响区域的分数
- **Alpha-Beta 剪枝：** 困难模式大幅减少搜索节点
- AI 计算时显示「思考中...」提示

### 左右玩家面板内容

- **回合指示条：** 当前玩家显示绿色「⬤ 轮到你了」持续闪烁；对手显示灰色「等待对手...」
- **棋子图标：** 大号渐变棋子（黑/白）
- **玩家名称：** 名字 + 角色（AI·困难 / 玩家）
- **倒计时：** 30 秒倒计时圆环，当前玩家红色闪烁，超时自动跳过回合
- **统计：** 落子数、总耗时

### 底部控制栏

- 模式切换：人机对战 / 双人对战
- AI 难度：简单 / 中等 / 困难
- 操作按钮：悔棋、新局、回放

## 视觉风格：传统木纹

- **棋盘背景：** Canvas 绘制木纹纹理（暖棕色渐变 + 噪点模拟）
- **网格线：** 深棕色半透明线条
- **黑棋：** 径向渐变（深灰 → 黑），模拟 3D 光泽
- **白棋：** 径向渐变（白 → 浅灰），细边框
- **最后落子标记：** 红色小圆点叠加
- **悬停预览：** 鼠标移到空位时显示半透明预览棋子
- **胜利动画：** 五连珠子闪烁 + 胜利弹窗
- **页面背景：** 深灰 #2d2d2d
- **面板/按钮主色：** 木质色调 #e8c99b

## 附加功能

### 悔棋

- 人机模式：撤回玩家 + AI 各一步（共两步）
- 双人模式：撤回上一步（对手刚下的那步），回合回到悔棋者
- 限制：只支持悔一步，不可连续悔棋

### 最后一步标记

- 最后落子位置叠加红色小圆点
- 新一步落下后标记自动移到新位置

### 棋局回放

- 游戏结束后可进入回放模式
- 提供「上一步 / 下一步 / 自动播放」控制
- 回放时棋盘逐步重现整局棋

### 状态信息栏

- 当前回合数
- 双方落子数
- 双方累计用时
- 倒计时显示

## 技术实现

### 项目结构

```
gomoku/
  └── index.html    # 单文件，内嵌 CSS + JS
```

纯前端单页应用，无需后端、无需构建工具、无需依赖库。

### 模块划分（逻辑分层）

| 模块 | 职责 |
|------|------|
| **Board** | 40×40 棋盘状态（二维数组）、落子、判胜、重置 |
| **Renderer** | Canvas 渲染（木纹、棋子、标记、动画） |
| **AI** | 启发式评分引擎、三档难度策略 |
| **GameController** | 游戏流程（模式切换、回合管理、悔棋、回放、倒计时） |
| **UI** | 玩家面板更新、按钮事件、名字输入弹窗 |

### 判胜算法

每次落子后，从该点出发沿 4 个方向检查是否形成五连。只需检查最后落子位置，无需遍历全盘。

### 倒计时

- 每回合 30 秒
- 当前玩家倒计时红色闪烁，对手倒计时静止
- 超时自动跳过该回合
- AI 回合倒计时正常显示（AI 通常秒内完成）

## 错误处理

- 点击已有棋子的位置：无响应
- 棋盘已满无人获胜：弹窗提示「平局」
- 超时跳过后如果连续超时：提示玩家是否继续
- AI 计算时间过长（>5 秒）：显示进度提示
