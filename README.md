# 地牢爬塔

暗黑奇幻风格的回合制文字爬塔游戏，提供两个版本：命令行版和网页版。

---

## 快速开始

### 网页版（推荐）

```bash
./web/play.sh
```

或者直接用浏览器打开 `web/index.html`。

### 命令行版

```bash
./play.sh
# 或
python main.py
```

> **环境要求**：Python 3.10+

## 游戏玩法

### 目标

爬过 **20 层**地牢，击败守卫，最终登顶成为地牢霸主。

### 职业选择

| 职业 | 血量 | 攻击 | 防御 | 特点 |
|------|------|------|------|------|
| 战士 | 150  | 25   | 15   | 血厚防高，稳扎稳打 |
| 法师 | 100  | 40   | 5    | 攻高血薄，爆发强 |
| 刺客 | 120  | 30   | 10   | 20% 暴击率，伤害×2 |

### 每层事件（随机3选1）

- **遭遇怪物**：必须战斗，胜利获金币，15% 概率掉落装备
- **宝箱**：随机获得金币 / 回血药水 / 装备
- **随机事件**：休息（+30HP）、陷阱（-20HP 且防御临时-5）、神秘商人

### 战斗系统

每回合三选一：

| 行动 | 效果 |
|------|------|
| 攻击 | 造成「你的攻击 - 怪物防御」点伤害，最低 1 |
| 防御 | 本回合受到的伤害减半 |
| 使用药水 | 回复 50 点血量 |

### 装备系统

装备分武器（加攻击）和防具（加防御）两类，各只能装备一件。

| 品质 | 武器加成 | 防具加成 |
|------|----------|----------|
| 普通（白）| +5 ~ 10  | +3 ~ 6   |
| 稀有（蓝）| +11 ~ 20 | +7 ~ 12  |
| 史诗（紫）| +21 ~ 30 | +13 ~ 20 |

### 升级机制

每击败 3 层怪物有 50% 概率升级：攻击和防御各 +5，最大血量 +20，血量回满。

## 玩法示例

### 示例一：战士硬刚流

战士血厚防高，适合莽撞推进。遇到弱怪无脑攻击，遇到高攻怪物时挂上防具再打。
碰到商人优先买防具，让防御超过怪物攻击力后每回合只掉 1 滴血。

```
第 8 层 — 遭遇「食人魔」ATK 20 / DEF 6
> 你的防御 18，每回合只受 2 点伤害
> 连续攻击 5 回合 → 食人魔倒下，获得 80 金
> 掉落 [稀有] 武器 · 烈火战斧 +17 攻击 → 拾取！
```

### 示例二：法师爆发流

法师攻击 40 起步，打低防怪物几乎秒杀。但血薄，遇到高攻怪物要先用药水垫血，
或者在「休息」格子回血后再冲。背包里至少备 2 瓶药水保命。

```
第 12 层 — 遭遇「暗影猎手」ATK 28 / DEF 11
> 法师攻击 58（Lv.3），每回合打出 47 点伤害
> 第 2 回合 → 暗影猎手倒下，几乎零损耗
第 13 层 — 遭遇「冰霜巨人」ATK 33 / DEF 17
> 伤害仅 41，但法师 HP 剩 42，扛不住
> 使用药水 → HP 回到 92 → 再打 3 回合胜利
```

### 示例三：刺客暴击流

刺客有 20% 暴击，伤害直接翻倍，运气好时一刀带走 Boss。
策略：一路攻击，遇到快被打死的情况用药水续命，等待暴击触发翻盘。
优先拾取武器装备，把攻击堆到最高，让暴击伤害尽可能爆炸。

```
第 16 层 — 遭遇「地牢守卫」ATK 36 / DEF 18
> 刺客攻击 52（装备+16），每回合打出 34 伤害
> 第 3 回合触发暴击！→ 造成 104 点伤害 → 直接秒杀
> *** 暴击！*** 获得 160 金 + [史诗] 武器掉落
```

### 示例四：商人屯货流

第 1 层碰到商人直接跳过，等打了几层金币够了再买。
优先买药水（30 金），把背包屯到 3 瓶，然后再买武器或防具。
遇到宝箱装备不急着装备，先放背包，等后面碰到更好的再决定。

```
第 5 层 — 神秘商人出现（当前金币 110）
> 买药水 ×2（60 金）→ 背包 3 瓶
> 买 [稀有] 防具 · 龙鳞甲 +11（50 金）→ 防御大涨
第 6 层 — 开宝箱
> 获得 [稀有] 武器 · 寒冰长剑 +14 → 拾取
> 装备 → 攻击再上一个台阶
```

### 关键生存法则

| 情景 | 建议操作 |
|------|---------|
| HP < 30% 且有药水 | 立刻用药水，别省着 |
| 怪物攻击远超防御 | 先防御一回合撑住，等机会 |
| 背包有更好装备 | 进入大厅时及时换装，别空跑 |
| 遇到陷阱（防御-5）| 本层遇怪优先防御行动 |
| 升级后血量回满 | 升级后是冲高层的最佳时机 |

## 网页版特效

- 暗黑奇幻主题，紫金霓虹配色
- 实体冲刺攻击动画，受击红闪抖动，死亡旋转消散
- 浮动伤害数字（暴击更大更亮）
- 血条平滑渐变（绿 → 黄 → 红）
- Canvas 粒子：升级金光爆炸、装备拾取紫光、暴击橙色粒子
- Web Audio API 合成音效，无需外部音频文件

## 项目结构

```
Tesseract/
├── main.py              # 命令行版入口
├── play.sh              # 命令行版启动脚本
├── game/                # 命令行版游戏逻辑
│   ├── character.py     # 角色与职业系统
│   ├── combat.py        # 回合制战斗
│   ├── dungeon.py       # 主游戏循环
│   ├── equipment.py     # 装备与背包
│   ├── events.py        # 非战斗事件
│   ├── monsters.py      # 怪物定义
│   └── utils.py         # 输入验证与工具函数
└── web/                 # 网页版
    ├── index.html       # 页面结构
    ├── style.css        # 主题样式与动画
    ├── play.sh          # 网页版启动脚本
    └── js/
        ├── audio.js     # Web Audio API 音效引擎
        ├── particles.js # Canvas 粒子特效
        ├── game.js      # 游戏逻辑（纯计算）
        ├── ui.js        # DOM 渲染与动画
        └── main.js      # 流程控制与事件绑定
```

## 扩展开发

代码中预留了扩展注释，以下位置可直接追加内容：

- **新增职业**：`game/character.py` → `CLASS_DEFINITIONS` / `web/js/game.js` → `CLASS_DATA`
- **新增怪物**：`game/monsters.py` → `MONSTER_POOL` / `web/js/game.js` → `MONSTER_TEMPLATES`
- **新增装备品质**：`game/equipment.py` → `QUALITY_TIERS` / `web/js/game.js` → `QUALITY_DATA`
- **新增随机事件**：`game/events.py` → `event_random()` 的 `events` 列表
